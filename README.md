# x86OS

A x86 (32 bits) operating system from scratch

## Description

### Project Structure

* nasm
* build-essential
* qemu-system
* mtools
* make

### Current State

#### Linux
```
sudo apt-get install nasm build-essential qemu-system mtools make
```

## Getting Started

### Prerequsites

* nasm
* build-essential
* qemu-system
* mtools
* make

### Installing

#### Linux
```
sudo apt-get install nasm build-essential qemu-system mtools make
```

### Executing x86OS

```
make run
```
